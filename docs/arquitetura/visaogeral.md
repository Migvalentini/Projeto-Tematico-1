# Visão Geral do Projeto

O *Budgetary Control Module* é um aplicativo desktop desenvolvido em Python com o objetivo de auxiliar usuários no gerenciamento de suas finanças pessoais de forma simples e prática. 
O sistema foi idealizado para atender pessoas que não possuem o hábito de organizar seus gastos e receitas, oferecendo uma solução acessível para o controle financeiro no dia a dia.

A aplicação permite o registro, categorização e acompanhamento de receitas e despesas, possibilitando uma melhor visualização da situação financeira do usuário e 
contribuindo para uma administração mais consciente dos recursos financeiros.

## MVP (Minimum Viable Product)

O MVP (*Minimum Viable Product*) do projeto contempla:

- Interface em linha de comando (terminal) para interação com o usuário;
- Funcionalidades de cadastro de rendas, despesas e categorias;
- Módulo de cálculo do orçamento mensal para acompanhamento financeiro;
- Integração com arquivos json para armazenamento e persistência dos dados cadastrados.

Com isso, o projeto busca promover maior organização financeira e oferecer aos usuários mais controle e liberdade em relação às suas finanças pessoais.
